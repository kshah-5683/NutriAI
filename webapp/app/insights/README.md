# Insights Route Segment (`webapp/app/insights/`)

This directory contains the user interface for tracking historical progress and macro distribution reports.

## 🎯 Major Function & Purpose

The `insights` route aggregates logged entry data over time and presents it visually using charting tools. Users can toggle through predefined tracking intervals (such as 7 days, 30 days, or a full year) to assess calorie intakes, macro splits, and target achievement success averages compared against custom goals.
