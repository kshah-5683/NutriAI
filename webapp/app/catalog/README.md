# Catalog Route Segment (`webapp/app/catalog/`)

This directory contains the user interface for managing custom food definitions.

## 🎯 Major Function & Purpose

The `catalog` route handles the display and search functionality of the user's personal food dictionary. It provides an interface to query existing food logs, launch custom editor sheets (for updates), add brand-new items, and remove retired foods from active lookup databases.
