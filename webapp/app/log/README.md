# Daily Log Route Segment (`webapp/app/log/`)

This directory contains the primary daily tracking screen for the nutrition application.

## 🎯 Major Function & Purpose

The `log` route serves as the central hub for the user's daily diary activities. It manages tracking for the selected date, displays macro progress rings against daily limits, lists logged foods group-by-meal slot, recommends personalized catalog/web meals, and provides triggers to log foods via natural language AI parsing, picture label scanning, or manual configurations.
