# Next.js Public Static Assets (`webapp/public/`)

This directory contains static assets (images, icons, vectors) served directly at the root of the web application.

## 🎯 Major Function & Purpose

The `public` folder in Next.js is used to store static files that do not need compilation or bundling. Files placed in this folder can be referenced directly from the root URL (e.g., `/vercel.svg`). It is primarily used to serve brand assets, global icons, favicons, and other static vector assets.
