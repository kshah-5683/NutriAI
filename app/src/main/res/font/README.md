# Typography Fonts (`app/src/main/res/font/`)

This directory houses the custom TrueType Font (TTF) files defining the typography of the Android application.

## 🎯 Major Function & Purpose

The `font` folder contains the typeface assets imported by the system to custom-style typography elements (headers, body text, labels) across all Jetpack Compose screens. Centralizing font assets here ensures consistent brand typography.
