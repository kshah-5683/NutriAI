# Launcher Icon Configurations (`app/src/main/res/mipmap-anydpi-v26/`)

This directory houses the adaptive launcher icon configuration files for devices running Android 8.0 (API 26) and above.

## 🎯 Major Function & Purpose

The `mipmap` folder coordinates launcher icons displayed on the device's home screen. The `-anydpi-v26` qualifier signifies that these XML adaptive icon resources scale dynamically for any screen density on API 26+ devices, combining separate foreground and background graphic layers to support custom launcher shapes and physics animations.
