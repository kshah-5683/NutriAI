# Graphic Drawables (`app/src/main/res/drawable/`)

This directory contains vector graphics, shapes, and image resource assets for the Android user interface.

## 🎯 Major Function & Purpose

The `drawable` folder organizes XML vector assets and image files rendered in UI layouts, buttons, dialog backdrops, and navigation icons. Using XML vectors ensures lossless scalability across varying screen densities and sizes.
