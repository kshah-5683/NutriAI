# Resource Configurations (`app/src/main/res/values/`)

This directory houses the XML configuration resources defining the static colors, localized text strings, and base styles/themes of the Android application.

## 🎯 Major Function & Purpose

The `values` folder stores simple XML key-value attributes (strings, integers, colors) and theme structures. Decoupling these constant values from the Kotlin codebase keeps the logic clean and enables quick edits to global theme profiles or localized texts.
