# Android App Resources (`app/src/main/res/`)

This directory houses the resource files (layouts, styles, drawables, fonts, strings) defining the visual presentation layer of the Android application.

## 🎯 Major Function & Purpose

The `res` folder organizes all XML and media assets parsed by the Android Asset Packaging Tool (AAPT). Separating these visual variables (like color values, string localizations, and vector drawings) from the Kotlin codebase enables responsive layout scaling, dynamic light/dark theming, and easy customization of asset files.
